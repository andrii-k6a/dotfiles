# Pi

Shared Pi resources.

## Setup

One option is to add this repo's resource paths to `~/.pi/agent/settings.json`:

```json
{
  "themes": ["~/path/to/dotfiles/pi/themes"],
  "extensions": ["~/path/to/dotfiles/pi/extensions"],
  "prompts": ["~/path/to/dotfiles/pi/prompts"],
  "skills": ["~/path/to/dotfiles/pi/skills"]
}
```
